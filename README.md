## Réplica da Interface do Spotify (Sem Funcionalidades)

Este repositório contém um projeto de front-end que replica a interface do usuário do Spotify, com foco na apresentação visual de fotos de artistas. Este projeto **não inclui funcionalidades de reprodução de música**.

## Objetivo

Demonstrar habilidades em desenvolvimento front-end (HTML, CSS e JavaScript) através da criação de uma interface similar à do Spotify. O projeto pode servir como base para estudos e experimentos com layouts responsivos, design de interface e manipulação de elementos visuais.

## Funcionalidades

  * Layout responsivo.
  * Estrutura da interface inspirada no Spotify.
  * Exibição de fotos de artistas em destaque.

## Tecnologias

  * HTML5
  * CSS3
  * JavaScript (opcional)

## Como Executar

1.  Clone este repositório.
2.  Abra o arquivo `index.html` em seu navegador.

## Observações

Este projeto foi desenvolvido como um exercício de front-end e não tem como objetivo substituir o Spotify ou infringir direitos autorais.
